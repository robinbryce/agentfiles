# scopes/forestrie

**Not the canonical source.** Team-shared Forestrie agent guidance lives in
[forestrie-agents](https://github.com/forestrie/forestrie-agents) — clone it
as a sibling under `~/Dev/personal/forestrie/` and read it first.

This folder holds only Robin's **personal** additions on top of that repo:
drafts not yet worth promoting to the team, and any purely personal
preferences that shouldn't go into shared team guidance at all.

## What lives here

| Path | Purpose |
|------|---------|
| [AGENTS.md](./AGENTS.md) | Router — points to forestrie-agents first |
| [incubator/](./incubator/) | Drafts destined for a forestrie-agents PR |

## Promotion

Anything here that's genuinely team-useful should be sanitized and PR'd into
`forestrie-agents` directly, following that repo's own promotion policy —
not merged into this repo's root `workflow/`.
